# 18.10 - CloudFormation - Intrinsic Functions

## Key Concepts

### 1. Full List of Intrinsic Functions
The most important ones (exam focus) are marked ⭐:

| Function | Purpose |
|----------|---------|
| `!Ref` ⭐ | Reference a parameter (returns value) or resource (returns physical ID) |
| `!GetAtt` ⭐ | Get a specific attribute of a resource |
| `!FindInMap` ⭐ | Retrieve a value from a Mapping |
| `!ImportValue` ⭐ | Import an exported value from another stack |
| `!Base64` ⭐ | Encode a string to Base64 (used for EC2 UserData) |
| `!If` / `!Not` / `!Equals` / `!And` / `!Or` ⭐ | Condition functions |
| `!Join` | Concatenate values with a delimiter |
| `!Sub` | Substitute variables in a string |
| `!Select` | Select one value from a list by index |
| `!Split` | Split a string into a list |
| `!GetAZs` | Get list of AZs in a region |
| `!Cidr` | Generate CIDR blocks |
| `!ForEach` | Iterate over a collection |
| `!ToJsonString` | Convert a value to a JSON string |
| `!Transform` | Apply a macro transform |
| `!Length` | Return the number of elements in a list |

### 2. !Ref
- On a **parameter** → returns the parameter's value
- On a **resource** → returns the physical ID of the resource (e.g. EC2 instance ID)

```yaml
VpcId: !Ref MyVPC   # MyVPC can be a resource or a parameter
```

### 3. !GetAtt
- Returns a **specific attribute** of a resource (beyond just its ID)
- Syntax: `!GetAtt ResourceName.AttributeName`
- What attributes are available → check the resource's documentation page under "Return values"

```yaml
# Get the AZ of an EC2 instance to place an EBS volume in the same AZ
AvailabilityZone: !GetAtt EC2Instance.AvailabilityZone
```

Common EC2 attributes: `AvailabilityZone`, `PrivateIp`, `PublicIp`, `PrivateDnsName`, `PublicDnsName`

### 4. !Base64
- Converts a string to Base64 encoding
- Primary use: passing **UserData** scripts to EC2 instances

```yaml
UserData:
  !Base64 |
    #!/bin/bash
    yum update -y
    yum install -y httpd
```

### 5. Ref vs GetAtt

| | `!Ref` | `!GetAtt` |
|--|--------|----------|
| Returns | Resource ID or parameter value | A specific attribute of a resource |
| Example | EC2 Instance ID | EC2 Public IP, AZ, DNS name |
| When to use | You need the ID | You need more details about the resource |

## Exam Tips
- `!Ref` on a resource → physical **ID**; on a parameter → **value**
- `!GetAtt` → use when you need something beyond the ID (IP, AZ, DNS, etc.)
- `!Base64` → almost exclusively used for EC2 **UserData**
- To know what `!GetAtt` can return → check the **"Return values"** section in the CloudFormation docs for that resource
- `!Join` / `!Sub` → useful for string building but not explicitly marked as exam-critical by Stephane
