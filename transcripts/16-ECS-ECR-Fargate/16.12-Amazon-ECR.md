# 16.12 - Amazon ECR

## Key Concepts

### 1. What is ECR?

- **Elastic Container Registry** — store and manage Docker images on AWS
- Alternative to Docker Hub, but private and integrated with AWS

### 2. Two Options

| Type | Description |
|------|-------------|
| **Private** | Only your account(s) can access |
| **Public** | Published to Amazon ECR Public Gallery |

### 3. Key Features

- Fully integrated with **ECS** (pull images directly)
- Images stored on **Amazon S3** behind the scenes
- Access controlled by **IAM** (permission error? check your policies)
- Supports: vulnerability scanning, versioning, image tags, image lifecycle

### 4. How ECR Works with ECS

```
ECR (stores Docker images)
  ↓ pull image (IAM role must allow)
ECS Cluster → EC2 Instance / Fargate → runs container
```
