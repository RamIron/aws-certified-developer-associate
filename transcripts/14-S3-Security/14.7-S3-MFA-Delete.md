# 14.7 - S3 MFA Delete

## Key Concepts

### What is MFA Delete?

Extra protection that forces users to provide an MFA code before performing destructive operations on versioned objects.

### When is MFA Required vs Not Required?

| MFA Required | MFA NOT Required |
|-------------|-----------------|
| Permanently delete an object version | Enable versioning |
| Suspend versioning on the bucket | List deleted versions |

### Requirements

- **Versioning must be enabled** on the bucket
- Only the **bucket owner (root account)** can enable/disable MFA Delete
- Can only be enabled/disabled via **CLI**, not from the AWS Console

---

## Exam Tips
- MFA Delete = protection against **permanent deletion** of object versions
- Only the **root account** can enable/disable it
- Can only be managed via **CLI** (not the AWS Console)
- Versioning must be enabled first
- Required for: permanent version delete + suspend versioning
