# 20.1 - AWS Monitoring - Section Introduction

> Udemy Lecture #235 · Section 20: AWS Monitoring & Audit: CloudWatch, X-Ray and CloudTrail

## Overview

Section intro — Stephane sets the stage for why monitoring matters:

- Your app is in the cloud, running fine... until your manager calls at 2 AM saying it's down
- Without monitoring enabled, you have no idea what happened
- Monitoring gives you: **logs**, **metrics**, **tracing**, and **audits** (who did what)
- As a developer, never deploy without enabling some kind of monitoring

## What This Section Covers

- **CloudWatch** — Metrics, Logs, Alarms, Synthetics
- **X-Ray** — Distributed tracing
- **CloudTrail** — API audit trail
- **EventBridge** — Event-driven reactions
