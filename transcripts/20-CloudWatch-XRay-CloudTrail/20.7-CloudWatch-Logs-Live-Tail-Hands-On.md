# 20.7 - CloudWatch Logs - Live Tail - Hands On

> Udemy Lecture #241 · Section 20: AWS Monitoring & Audit: CloudWatch, X-Ray and CloudTrail

## Hands-On Summary

### Live Tail Feature

- Real-time log viewer — logs appear as they are posted
- Navigate: Log Group → click **Start Tailing**
- Filter by: log group (required), log stream (optional), filter pattern (optional)
- Events stream into the UI as they happen — great for **live debugging**
- Click on any event to get details (timestamp, group, link to log stream)

### Pricing

- **~1 hour/day free** usage of Live Tail
- Make sure to **stop/close the Live Tail session** when done to avoid charges

### Live Tail vs Logs Insights

| | Live Tail | Logs Insights |
|--|-----------|---------------|
| **When** | Real-time (as events arrive) | Historical (query past data) |
| **Use case** | Debugging live | Analyzing patterns |
| **Pricing** | ~1 hour/day free | Per query |
