# 20.26 - CloudTrail Hands On

## Overview
Hands-on demo showing how to view API call history in the CloudTrail console.

## Key Takeaways

### Event History
- CloudTrail console → left panel → **Event History**
- Shows last **90 days** of **management events**
- Every API call is recorded: who, what, when, from where

### Demo: Terminate EC2 Instance
1. Terminated a demo EC2 instance from the console
2. Waited ~5 minutes for the event to appear in CloudTrail
3. Found `TerminateInstances` event in CloudTrail with:
   - **Event source:** EC2
   - **Access key** used
   - **Region**
   - Full **event JSON** with all details

### Key Observation
- Events take a few minutes to appear (not instant)
- The full JSON event record contains all API call details

## Exam Tips
- CloudTrail Event History = last 90 days of management events in the console
- Events may take a few minutes to appear after the action
