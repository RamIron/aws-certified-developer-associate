# 19.15 - SNS Hands On

## Exam-Relevant Observations

### 1. SNS FIFO Topic — Subscriber Limitation
- FIFO SNS topic can **only** have **SQS queues** as subscribers (no Lambda, Email, HTTP, etc.)
- Standard SNS topic supports all subscriber types: SQS, Lambda, KDF, Email, SMS, HTTP/HTTPS

### 2. Email/SMS Subscription Confirmation
- **Email and SMS** subscriptions start as **"Pending confirmation"**
- Subscriber must click a confirmation link before messages are delivered
- AWS service subscribers (SQS, Lambda, KDF) are confirmed automatically — no manual step needed

### 3. SNS Access Policy Recap
- Works like S3 bucket policies / SQS queue access policies
- Defines who can **publish** to the topic
- Needed when allowing AWS services (e.g., S3) to write events to SNS
